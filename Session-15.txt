Special variables in shell
============================
we no need to define the variable, shell will provide to us.


if you want all variables passed to script: $@
number of variables passed: $#
script name: $0
present working directory: $PWD (capital letters)
home directory of the user who is running the script: $HOME
which user is running the script: $USER
PID of the current script: $$
sleep 10 & --> goto background in 10 seconds
PID of the last command running in background: $!


we want run this specail variables in script.

#!/bin/bash

USERID=$(id -u)

if [ $USERID -ne 0 ]
then
    echo "ERROR:: Please run this script with root access"
    exit 1 #give other than 0 upto 127
else
    echo "You are running with root access"
fi

# validate functions takes input as exit status, what command they tried to install
VALIDATE(){
    if [ $1 -eq 0 ] --> ( $1 = $? (exit status of previous command):  $2 = "MySQL")
    then
        echo "Installing $2 is ... SUCCESS"
    else
        echo "Installing $2 is ... FAILURE"
        exit 1  #If installation fails:It stops the script immediately.
    fi
}

dnf list installed mysql  #Checks if mysql package is installed.
if [ $? -ne 0 ]  #exit status of previous command not equal 0: If mysql is NOT installed (exit code not 0)
then
    echo "MySQL is not installed... going to install it"
    dnf install mysql -y
    VALIDATE $? "MySQL" #VALIDATE function prints result
else
    echo "MySQL is already installed...Nothing to do" # print already installed
fi

dnf list installed python3
if [ $? -ne 0 ]
then
    echo "python3 is not installed... going to install it"
    dnf install python3 -y
    VALIDATE $? "python3"
else
    echo "python3 is already installed...Nothing to do"
fi

dnf list installed nginx
if [ $? -ne 0 ]
then
    echo "nginx is not installed... going to install it"
    dnf install nginx -y
    VALIDATE $? "nginx"
else
    echo "nginx is already installed...Nothing to do"
fi


##### "This script checks whether MySQL, Python3, and Nginx are installed.
If not installed, it installs them.
After installation, it validates whether the installation was successful using a reusable function." #####
