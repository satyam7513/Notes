3️.Waterfall Model:
==================
Earlier software was developed using Waterfall Model.

"In Waterfall, each phase must be completed fully before moving to the next phase."

Example timeline:

Requirement → 3 months
Design → 3 months
Development → 8 months
Testing → 6 months
Deployment → 4 months

Total → 2 years."

"If after 2 years the customer says:  'I want a new feature.'

What will happen?"

"Everything must be redesigned. Very expensive."

example: 100 defects during testing
           20 invalid defects
		   
Key points:

No going back easily

Changes are difficult

Everything is planned at the beginning.



4.Agile Model :
===============
"So companies moved to Agile Model."

"Instead of building software for 2 years, we divide work into small cycles called Sprints."

Sprint duration: 2 weeks

Sprint process:  Plan  Develop  Test  Deploy

Example: E-commars site like meesho

Sprint 1 → Login feature
Sprint 2 → Orders feature
Sprint 3 → Payments


Advantages:

✔ Faster delivery
✔ Customer feedback early
✔ Less risk


Waterfall
1000 tests → 100 defects

Agile
100 tests → 3 defects

But Agile still has one problem: 

"Developers write code fast…  But deployment is still slow."

Then introduce DevOps.


5.DevOps Concept:
=================

Dev + Ops = DevOps

Explain:

Developers write code.

Operations team deploys and manages servers.

Earlier problem:

Developer → "My code works"
Operations → "Application not working" -- No collaboration.

DevOps solves this.

"DevOps is a culture and practice that combines Development and Operations to deliver software faster 
using automation."

Continuous Integration
Continuous Delivery
Continuous Deployment
Continuous Testing
Automation

Developer writes code
↓
Push to Git
↓
Jenkins builds code
↓
Automated testing
↓
Docker image created
↓
Deploy to Kubernetes
↓
Application live


Example: Earlier deployment time: 6 months

with Devops Deployment time: 10–50 deployments per day.
