What is Forward Proxy?
===========================

A forward proxy sits between client (user) and internet servers.

👉 Client sends request → Proxy → Internet server
👉 Server response → Proxy → Client

The server does NOT know the real client.


Why Use Forward Proxy?

Hide client identity (privacy)

Restrict websites (company/school filtering)

Cache data to improve speed

Security monitoring


Real Life Example:
====================

🏫 College / Office Network

Students try to open websites

All traffic goes through proxy server

Admin can block YouTube, social media etc.



What is Reverse Proxy?

A reverse proxy sits between internet users and backend servers.

👉 Client → Reverse Proxy → Backend Servers
👉 Servers → Reverse Proxy → Client

Client does NOT know actual server.


Why Use Reverse Proxy?

Load balancing

Security protection

SSL termination

Hides internal servers

Improves performance


Real Life Example:
===================

🌐 Big Websites (Amazon, Netflix, Flipkart)

Many backend servers exist

Reverse proxy distributes traffic

If one server fails → sends request to another
